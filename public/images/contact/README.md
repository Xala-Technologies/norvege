# Contact Page Hero Image

Place your hero image for the Contact page here.

**File name:** `hero.jpg`

**Recommended specifications:**

- Format: JPG or PNG
- Dimensions: 1920x1080px or larger (16:9 aspect ratio recommended)
- File size: Optimized for web (under 500KB recommended)
- Content: Should represent contact/communication theme, professional and relevant to mineral exploration

The image will be displayed as a full-background hero section with text overlaid on top.
